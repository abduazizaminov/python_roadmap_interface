Recursion - Рекурсия

Рекурсия дар барномасозӣ ин қобилияти муайян кардани функсия бо истифода аз худи функсияи муайяншуда дар раванд мебошад. Дар математика бисёр функсияҳо бо ин роҳ муайян карда мешаванд, аз ин рӯ аксари забонҳои барномасозӣ ин равишро истифода мебаранд. Ин дар Python низ кор мекунад. Одатан, барои муяайянкунии функсия, маълумоте, ки пеш дода шуда буд истифода бурда мешавад. Аммо як истисно вуҷуд дорад - функсия дар бадани он метавонад худро даъват кунад. Чунин намудро дорад: 

        def factorial(n):
    if n <= 0:
        return 1
    return n * factorial(n - 1)
Ин функсия факториали ададро n-hj тавассути зарб кардани адад ба факториали адади n - 1 ҳисоб мекунад.

Шарти ба итмомрасонидани рекурсия

Мисоли боло шартеро истифода мебарад, ки рекурсияро бозмедорад. Агар аз функсия шартеро, ки аргументи манфиро тафтиш мекунад, хориҷ карда шавад, он гоҳ ҳангоми клик кардани ин функсия барномаро аз кор мебарорад ё функсия такроран худашро даъват мекунад.

Чаро рекурсия лозим аст?

Баъзе алгоритмҳо тезтар татбиқ мешаванд, агар баҷои сикл рекурсия истифода бурда шавад. . Ҳангоми татбиқи чунин алгоритмҳо, шумо бояд дар хотир доред, ки хотираи стек маҳдуд аст. Дар ин ҳолат, худи сохторҳои додаҳо одатан маҳдуданд, бинобар ин шумо набояд аз рекурсия комилан даст кашед.
    Хусусиятҳои рекурсия:
  • Иҷрои як амалиёт якчанд маротиба бо воридоти гуногун.
  • Дар ҳар як қадам, мо кӯшиш мекунем, ки саҳмияҳои хурдтар гузорем, то мушкилотро хурдтар созем.
  • Барои қатъ кардани рекурсия шарти асосӣ лозим аст, вагарна гардиши беохир ба амал меояд.
      • Намудҳои рекурсия
    Рекурсияро вобаста ба тарзи даъват кардан ба 2 намуд ҷудо карда мешавад.
  • Бевосита - вақте ки функсия худашро даъват мекунад
  • Бавосита - ҳангоми як функсия дар дохили худ функсияи дигарро даъват кунад ва функсияи пешинаро даъват кунад
    • екурсияҳоро инчунин аз рӯи шумораи даъватҳои рекурсивӣ тақсим кардан мумкин аст:
  • Хатӣ - вақте ки ҳангоми ҳисоб кардани натиҷаи функсия, функсия худро як маротиба даъват мекунад, тавре ки дар мисоли factorial. Биёед фаҳмонем, ки "як бор" маънои шумораи умумии даъватҳои функсионалии танаро надорад. Сухан дар бораи шумораи даъватҳое меравад, ки натичахои онхо барои як хисоби умумй заруранд
  • Каскад - вақте ки функсия якчанд маротиба худро даъват мекунад

    • Мисоли рекурсияи хатӣ

    Агар рекурсия дар функсия гипотезаи Collatz-ро санҷида бошад , он хатӣ ҳисобида мешавад : 
    
    def collatz(n):
    if n == 1:
        return True
    if n % 2 == 0:
        return collatz(n // 2)
    return collatz(n * 3 + 1)
    Дар ин ҷо дар тани функсия ду намуди даъвати рекурсивӣ мавҷуд аст, аммо дар ҳар як даъват танҳо як даъват истифода мешавад. Мисоли дигари истифодаи рекурсияи хатӣ ин гузариш аз коллексияҳо мебошад. Барои ин, коллексияро ба таври рекурсивӣ чунин муаррифӣ карда мешавад:

    Намунаи рекурсияи каскади

    Агар рекурсия дар функсия бо Адади Фибоначчи ҳисоб карда шавад, он каскади ҳисобида мешавад: 
    
    def fibonacci(n):
    if n <= 2:
        return 1
    return fibonacci(n - 1) + fibonacci(n - 2)

Дар инҷо функсия ҳамеша худро ду маротиба даъват мекунад.
Дар аввал ду маротиба даъват мешавад, ки ба чор маротиба табдил меёбад. Яъне ду даъват ба ду маротиба  табдил меёбад. Пас аз ҳашт - шумораи зангҳо каскад меафзояд
