Sorting Algorithms - Ҷобаҷокунииалгоритмҳо

Ҷобаҷокунӣ ҳамчун ҷойгиркунии маълумот бо тартиби муайян муайян карда мешавад. Усулҳои ҷудокунӣ барои тартиб додани маълумот (асосан ададӣ) бо тартиби афзоиш ё камшавӣ истифода мешаванд. Ин усулест, ки муаррифии маълумот дар формати бештар фаҳмо истифода мешавад. Ин як соҳаи муҳими илмҳои компютерӣ мебошад. Ба навъбандии миқдори зиёди маълумот метавонад миқдори зиёди захираҳои ҳисоббарориро талаб кунад, агар усулҳое, ки мо барои ҷудо кардани маълумот истифода мебарем, бесамар бошанд. Самаранокии алгоритм ба шумораи ашёҳое, ки аз он мегузарад, мутаносиб аст. Барои миқдори ками маълумот, усули мураккаби ҷудокунӣ метавонад аз арзиши он мушкилтар бошад. Аз тарафи дигар, барои миқдори зиёди маълумот, мо мехоҳем то ҳадди имкон самаранокӣ ва суръатро зиёд кунем. Мо ҳоло якчанд усулҳои ҷудокуниро баррасӣ мекунем ва онҳоро аз рӯи мураккабии вақт муқоиса мекунем.
    Усулҳои ҷудокунӣ
Татбиқи гуногуни усулҳои ҷудокунӣ дар Python инҳоянд:
  • Тартиби ҳубобӣ
  • Тартиби интихоб
  • Тартиби воридкунӣ

    • Тартиби ҳубобӣ

    Bubble Sort як алгоритми оддии ҷудокунӣ мебошад. Ин алгоритми ҷудокунӣ ду унсури монандро такроран муқоиса мекунад ва агар онҳо дар тартиби нодуруст бошанд, онҳоро иваз мекунад. Ба ибораи дигар, мо ду унсури ҳамсояро муқоиса мекунем ва мебинем, ки тартиби онҳо нодуруст аст, агар фармоиш нодуруст бошад, мо онҳоро иваз мекунем. (яъне arr[i] > arr[j] барои 1 <= i < j <= s; дар он ҷо s андозаи массив аст, агар массив бояд бо тартиби афзоиш бошад ва баръакс).

    Мисол 

    
    # Python3 program for Bubble Sort Algorithm Implementation
def bubbleSort(arr):
	
	n = len(arr)

	# For loop to traverse through all 
	# element in an array
	for i in range(n):
		for j in range(0, n - i - 1):
			
			# Range of the array is from 0 to n-i-1
			# Swap the elements if the element found 
			#is greater than the adjacent element
			if arr[j] > arr[j + 1]:
				arr[j], arr[j + 1] = arr[j + 1], arr[j]
				
# Driver code

# Example to test the above code
arr = [ 2, 1, 10, 23 ]

bubbleSort(arr)

print("Sorted array is:")
for i in range(len(arr)):
	print("%d" % arr[i])

    Тартиби интихоб

    Ин техникаи људобандї такроран элементи минималиро пайдо мекунад ва онро бо тартиб људо мекунад Bubble Sort ягон фазои иловагии хотираро ишғол намекунад. Ҳангоми иҷрои ин алгоритм, ду зермассиве нигоҳ дошта мешавад, зермассиви аллакай ҷудошуда ва зермассиви боқимондаи ҷудонашуда. Ҳангоми иҷроиши Интихоби Тартиб барои ҳар як такрор, элементи ҳадди ақали зермассиви ҷудонашуда дар зермассиви ҷудошуда ҷойгир карда мешавад. Навъи интихоб як алгоритми самараноктар аз навъбандии ҳубобӣ аст

    Мисол

    
    # Selection Sort algorithm in Python
def selectionSort(array, size):
	
	for s in range(size):
		min_idx = s
		
		for i in range(s + 1, size):
			
			# For sorting in descending order
			# for minimum element in each loop
			if array[i] < array[min_idx]:
				min_idx = i

		# Arranging min at the correct position
		(array[s], array[min_idx]) = (array[min_idx], array[s])

# Driver code
data = [ 7, 2, 1, 6 ]
size = len(data)
selectionSort(data, size)

print('Sorted Array in Ascending Order is :')
print(data)
    Натиҷа 
    
    Sorted Array in Ascending Order is: 
[1, 2, 6, 7]
    Тартиби воридкунӣ Ин алгоритми ҷудокунӣ зер-массивро нигоҳ медорад, ки ҳамеша ба навъҳоҷудо карда мешавад. Қиматҳо аз қисми ҷудонашудаи массив дар мавқеи дуруст дар қисми ҷудошуда ҷойгир карда мешаванд. Он дар амал нисбат ба дигар алгоритмҳо, ба монанди навъҳои интихоб ё навъбандии ҳубобӣ самараноктар аст.

    Амалисозии навъбандии воридкунӣ

    
    # Creating a function for insertion sort algorithm
    def insertion_sort(list1): 
    
            # Outer loop to traverse on len(list1) 
            for i in range(1, len(list1)): 
    
                a = list1[i] 
    
                # Move elements of list1[0 to i-1], 
                # which are greater to one position
                # ahead of their current position 
                j = i - 1
            
                while j >= 0 and a < list1[j]: 
                    list1[j + 1] = list1[j] 
                    j -= 1
                    
                list1[j + 1] = a 
                
            return list1 
                
    # Driver code
    list1 = [ 7, 2, 1, 6 ] 
    print("The unsorted list is:", list1) 
    print("The sorted new list is:", insertion_sort(list1))
    Натиҷа 
    
        The unsorted list is: [7, 2, 1, 6]
        The sorted new list is: [1, 2, 6, 7]
