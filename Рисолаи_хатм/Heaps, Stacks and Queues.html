Heaps, Stacks and Queues - Стекҳо, тӯдаҳо ва навбатҳо

Heaps in Python Сохтани тӯда Тӯда бо истифода аз китобхонаи дохилии python бо номи heapq сохта мешавад. Ин китобхона дорои вазифаҳои мувофиқ барои анҷом додани амалиётҳои гуногун дар сохтори маълумотҳои теппа мебошад. Дар зер рӯйхати ин функсияҳо оварда шудааст.
  • heapify - Ин функсия рӯйхати муқаррариро ба тӯда табдил медиҳад. Дар тӯдаи натиҷа хурдтарин элемент ба мавқеи индекс 0 тела дода мешавад. Аммо боқимондаи элементҳои додаҳо ҳатман мураттаб карда намешаванд.
  • heappush - Ин функсия бе тағир додани тӯдаи ҷорӣ элементро ба теппа илова мекунад.
  • heappop - Ин функсия хурдтарин унсури маълумотро аз теппа бармегардонад.
  • heapreplace - Ин функсия хурдтарин унсури маълумотро бо арзиши наве, ки дар функсия дода шудааст, иваз мекунад.
Сохтани тӯдаи одди
  • Heapify (iterable): - Ин функсия барои табдил додани объекти такроршаванда дар сохтори додаҳои тӯда истифода мешавад. 
    
        # importing "heapq" to implement heap queue
import heapq

# initializing list
li = [5, 7, 9, 1, 3]

# using heapify to convert list into heap
heapq.heapify(li)

# printing created heap
print ("The created heap is : ",(list(li)))
Натиҷа
Тӯдаи сохташуда: [1, 3, 9, 7, 5]
    • Дохилкуни ва хориҷкунии элемент
  • heappush(heap, ele) : Ин функсия барои ворид кардани унсури дар аргументҳои он зикршуда ба тӯда истифода мешавад. Тартиб танзим карда мешавад, то сохтори теппа нигоҳ дошта шавад.
  • heappop(heap) : Ин функсия барои нест кардан ва баргардонидани элементи хурдтарин аз тӯда истифода мешавад. Тартиб тавре танзим карда мешавад, ки сохтори тӯда нигоҳ дошта шавад. 
    • 
        # importing "heapq" to implement heap queue
import heapq

# initializing list
li = [5, 7, 9, 1, 3]

# using heapify to convert list into heap
heapq.heapify(li)

# printing created heap
print("The created heap is : ", end="")
print(list(li))

# using heappush() to push elements into heap
# pushes 4
heapq.heappush(li, 4)

# printing modified heap
print("The modified heap after push is : ", end="")
print(list(li))

# using heappop() to pop smallest element
print("The popped and smallest element is : ", end="")
print(heapq.heappop(li))
Натиҷа
The created heap is : [1, 3, 9, 7, 5]
The modified heap after push is : [1, 3, 4, 7, 5, 9]
The popped and smallest element is : 1
    Илова ва хориҷ кардан дар як вақт
  • heappushpop(heap, ele) :- Ин функсия амалиёти ҳам push ва pop-po дар як изҳорот муттаҳид намуда, самаранокии онро зиёд мекунад. Тартиби теппа пас аз ин амалиёт нигоҳ дошта мешавад.
  • heapreplace(heap, ele) :- Ин функсия инчунин элементҳоро дар як оператор дохил мекунад ва мебарорад. Дар ин, элемент аввал дохил карда мешавад, баъд элемент хориҷ мекунад, яъне, арзиши калонтар аз арзиши дохилкарда метавонад баргардонида шавад. heapreplace() дар муқоиса бо heappushpop() арзиши хурдтаринро дар тӯда бар мегардонад, новобаста аз элементи дохилшуда. 
    • 
        # importing "heapq" to implement heap queue
import heapq

# initializing list 1
li1 = [5, 1, 9, 4, 3]

# initializing list 2
li2 = [5, 7, 9, 4, 3]

# using heapify() to convert list into heap
heapq.heapify(li1)
heapq.heapify(li2)

# using heappushpop() to push and pop items simultaneously
# pops 2
print("The popped item using heappushpop() is : ", end="")
print(heapq.heappushpop(li1, 2))

# using heapreplace() to push and pop items simultaneously
# pops 3
print("The popped item using heapreplace() is : ", end="")
print(heapq.heapreplace(li2, 2))
Натиҷа
The popped item using heappushpop() is : 1
The popped item using heapreplace() is : 3

    Стек дар Python

    Стек сохтоҷри хаттии додаҳост, ки элементро ба тарзи Last-In/First-out (LIFO) ё First-In/Last-out (FILO) нигоҳ медорад. Дар стек, як элементи нав дар як канор илова карда мешавад ва элемент танҳо аз он канор хориҷ карда мешавад. Амалҳои дохил кардан ва нест карданро аксар вақт push ва pop меноманд.
      Функсияҳое, ки бо стек алоқаманданд:
    • empty() – бармегардонад, ки оё холӣ будани стек
    • size() – андозаи стекро бармегардонад
    • top() / peek() – истинодро ба унсури болоии стек бармегардонад
    • push(a) - элементи 'a'-ро дар болои стек ҷойгир мекунад
    • pop() – элементи болотарини стекро нест мекунад
    Стеки Python метавонад бо истифода аз синфи deque аз модули коллексияҳо амалӣ карда шавад. Дар ҳолатҳое, ки ба мо амалиёти зудтари замима ва поп аз ҳарду канори контейнер лозим аст, аз рӯйхат бартарӣ дода мешавад, зеро deque дар муқоиса бо рӯйхате, ки O(n) -ро таъмин мекунад, мураккабии вақти O(1)-ро барои амалиёти замима ва поп таъмин мекунад. мураккабии вақт. Ҳамин усулҳое, ки дар рӯйхат дида мешаванд, дар deque истифода мешаванд, append() ва pop(). 
    
    # Python program to
# demonstrate stack implementation
# using collections.deque

from collections import deque

stack = deque()

# append() function to push
# element in the stack
stack.append('a')
stack.append('b')
stack.append('c')

print('Initial stack:')
print(stack)

# pop() function to pop
# element from stack in
# LIFO order
print('\nElements popped from stack:')
print(stack.pop())
print(stack.pop())
print(stack.pop())

print('\nStack after elements are popped:')
print(stack)

# uncommenting print(stack.pop())
# will cause an IndexError
# as the stack is now empty
