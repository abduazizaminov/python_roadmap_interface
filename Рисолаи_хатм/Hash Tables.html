Hash Tables - Хеш ҷадвалҳо

Ҷадвали хэш ё харитаи хэш дар Python Ҷадвалҳои хэш як намуди сохтори додаҳо мебошанд, ки дар он суроға ё арзиши индекси элементи додаҳо аз функсияи ҳаш генератсия мешавад. Ин дастрасии маълумотро тезтар мекунад, зеро арзиши индекс ҳамчун калиди арзиши додаҳо амал мекунад. Ба ибораи дигар, ҷадвали Hash ҷуфтҳои калид-арзишро нигоҳ медорад, аммо калид тавассути функсияи hasshing сохта мешавад.
Ҳамин тавр, функсияи ҷустуҷӯ ва воридкунии элементи додаҳо хеле тезтар мешавад, зеро худи арзишҳои калидӣ ба индекси массиве табдил меёбанд, ки маълумотро нигоҳ медорад.
Дар Python, намудҳои маълумоти Dictionary татбиқи ҷадвалҳои хэшро ифода мекунанд. Калидҳои луғат ба талаботи зерин ҷавобгӯ мебошанд.
  • Калидҳои луғат хэшшавандаанд, яъне онҳо тавассути функсияи хэшинг сохта мешаванд, ки барои ҳар як арзиши уникали, ки ба функсияи hash дода мешавад, натиҷаи уникали месозад.
  • Тартиби элементҳои додаҳо дар луғат муқаррар карда нашудааст.

    • Дастрасӣ ба арзишҳо дар луғат

    Барои дастрасӣ ба элементҳои луғат, шумо метавонед қавсҳои мураббаъро дар якҷоягӣ бо калид истифода баред, то арзиши онро ба даст оред.
    Мисол 
    
    # Declare a dictionary 
dict = {'Name': 'Zara', 'Age': 7, 'Class': 'First'}

# Accessing the dictionary with its key
print ("dict['Name']: ", dict['Name'])
print ("dict['Age']: ", dict['Age'])
    Натиҷа 
    
    dict['Name']:  Zara
dict['Age']:  7

    Навсозии луғат

    Шумо метавонед луғатро тавассути илова кардани вуруди нав ё ҷуфти калид-арзиш, тағир додани вуруди мавҷуда ё нест кардани вуруди мавҷуда тавре, ки дар мисоли оддӣ дар зер нишон дода шудааст, навсозӣ кунед
    Мисол 
    
    # Declare a dictionary
dict = {'Name': 'Zara', 'Age': 7, 'Class': 'First'}
dict['Age'] = 8; # update existing entry
dict['School'] = "DPS School"; # Add new entry
print ("dict['Age']: ", dict['Age'])
print ("dict['School']: ", dict['School'])

    Натиҷа

    
dict['Age']:  8
dict['School']:  DPS School

    Унсурҳои луғатро нест кунед

    Шумо метавонед унсурҳои луғатро хориҷ кунед ё тамоми мундариҷаи луғатро тоза кунед. Шумо инчунин метавонед тамоми луғатро дар як амалиёт тоза кунед.

    Мисол

    
dict = {'Name': 'Zara', 'Age': 7, 'Class': 'First'}
del dict['Name']; # remove entry with key 'Name'
dict.clear();     # remove all entries in dict
del dict ;        # delete entire dictionary

print ("dict['Age']: ", dict['Age'])
print ("dict['School']: ", dict['School'])

    Натиҷа

    
dict['Age']:  dict['Age']
dict['School']:  dict['School']
