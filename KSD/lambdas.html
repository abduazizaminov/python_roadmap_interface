Lambdas - Лямбдаҳо

Функсияи ламбда (ё "функсияи анонимӣ") - намуди махсуси функсия бе ном аст. Масалан:


        lambda : print('Hello, World!')

Дар ин ҷо мо функсияи ламбдаро сохтем, ки матни Hello, World!-ро мебарорад.

Эълон кардани функсияи ламбда дар Python

Калимаи калидии lambda (ба ҷои def) барои сохтани функсияи ламбда истифода мешавад. Синтаксиси эълон кардани функсияи ламбда:


        lambda аргумент(ҳо) : ифода

Дар ин ҷо:

  • аргумент(ҳо) - ҳама гуна арзише, ки ба функсияи ламбда дода мешавад;

  • ифода - ин изҳороте, ки иҷро ва баргардонида мешавад.

Мисол дида мебароем:


        greet = lambda : print('Hello, World!')

Дар ин ҷо мо функсияи ламбдаро муайян кардем, ки онро ба тағирёбандаи greet таъин кардем.

Барои иҷрои ин функсияи ламбда, мо бояд онро даъват кунем. Ин аст, ки чӣ тавр мо метавонем функсияи ламбдаро даъват намоем:


        # Фарёд кардани функсияи ламбда
        greet()

Функсияи ламбда, ки дар боло овардашуда буд, матни Hello, World!-ро мебарорад.

Мисоли даъват кардани функсияи ламбдаро дар Python дида мебароем:


        # Эълон кардани функсияи ламбда
        greet = lambda : print('Hello, World!')
        
        # Фарёд кардани функсияи ламбда
        greet()

Натиҷа:


        Hello, World!

Дар ин ҷо мо функсияи ламбдаро муайян кардем ва онро ба тағирёбандаи greet додем. Вақте ки мо функсияи ламбдаро фарёд мекунем, функсияи print() дар дохили функсияи ламбда иҷро мешавад.

Функсияи ламбда бо аргументҳо дар Python

Мисли функсияҳои муқаррарӣ, функсияи ламбда низ метавонад аргументҳоро қабул кунад. Масалан:


        # Функсияи ламбда, ки 1 аргумент мегирад
        greet_user = lambda name : print('Hey there,', name)
        
        # Фарёд кардани функсияи ламбда
        greet_user('Delilah')

Натиҷа:


        Hey there, Delilah

Дар ин ҷо мо вазифаи ламбдаро ба тағирёбандаи greet_user таъин кардем. name пас аз калимаи калидии ламбда нишон медиҳад, ки функсияи ламбда аргументро бо номи name мегирад.

Ба фарёд кардани функсияи ламбда диққат диҳед:


        greet_user('Delilah')

Дар ин ҷо мо арзиши сатри Delilah-ро ба функсияи лямбдаамон, ки ба аргументи name мувофиқат мекунад, гузоштем.

Истифодабарии функсияи ламбда бо filter()

Функсияи filter() дар Python ҳамчун аргумент функсия, рӯйхатҳо, наворҳо ва сатрҳоро мегирад. Функсия бо ҳамаи элементҳои рӯйхат даъват карда мешавад ва рӯйхати наве бармегардад, ки дорои элементҳое мебошад, ки функсия барои онҳо True муайян кардааст.

Масалан:


        # Барномаи филтри элементҳои ҷуфти рӯйхат
        my_list = [1, 5, 4, 6, 8, 11, 3, 12]
        
        new_list = list(filter(lambda x: (x%2 == 0) , my_list))
        
        print(new_list)

Натиҷа:


        [4, 6, 8, 12]

Дар ин ҷо функсияи filter() танҳо рақамҳои ҷуфтро аз рӯйхат бармегардонад.

Истифодабарии функсияи ламбда бо map()

Функсияи map() дар Python ҳамчун аргумент функсия, рӯйхатҳо, наворҳо ва сатрҳоро мегирад. Функсия бо ҳамаи элементҳои рӯйхат даъват карда мешавад ва рӯйхати наве бармегардад, ки дорои элементҳои аз ҷониби ин функсия барои ҳар як элемент баргардонидашуда мебошад.

Масалан:


        # Барнома барои дучанд кардани арзиши ҳар як элементи рӯйхат бо истифода аз функсияи map()
 
        my_list = [1, 5, 4, 6, 8, 11, 3, 12]
        
        new_list = list(map(lambda x: x * 2 , my_list))
        
        print(new_list)

Натиҷа:


        [2, 10, 8, 12, 16, 22, 6, 24]

Дар ин ҷо функсияи map() арзишҳои элементҳои рӯйхатро дучанд мекунад.
