PyPI - Python Package Index

PyPI, ё Python Package Index, як анбори марказӣ барои бастаҳоест, ки бо истифода аз забони барномасозии Python таҳия шудааст. Он як анбори онлайн пешниҳод мекунад, ки дар он таҳиягарон метавонанд библиотекаҳо, модулҳо, фреймворкҳо ва асбобҳои худро нашр кунанд, то корбарони дигари Python онҳоро ба осонӣ дар лоиҳаҳои худ пайдо, насб ва истифода баранд.

Насб кардани PyPI

Барои насб кардани PyPI (Index Package Python), шумо бояд Python насб карда бошед, зеро PyPI анбори стандартӣ барои бастаҳои Python аст. Дар аксари ҳолатҳо, Python дорои PIP аст, воситаи идоракунии бастаҳо, ки барои зеркашӣ ва насб кардани бастаҳо аз PyPI истифода мешавад.

Барои насб кардани PyPI (Python Package Index):

  1. Насб кардани Python:

    • Насбкунандаи Python-ро аз вебсайти расмии Python зеркашӣ кунед.
    • Насбкунандаро иҷро кунед ва дастурҳоро риоя кунед.
    • Боварӣ ҳосил кунед, ки "Add Python to PATH" дар ҷараёни насб интихоб шудааст.

  2. Санҷиши мавҷудияти PIP:

    • Сатри фармонро кушоед (дар Windows ин корро тавассути пахш кардани Win + R, ворид кардани "cmd" ва пахш кардани Enter анҷом додан мумкин аст).
    • Дар сатри фармон, pip--version -ро ворид кунед. Агар pip насб карда шуда бошад, шумо маълумоти версияи pip-ро мебинед.

  3. Навсозии PIP:

    • Дар сатри фармон, python-m pip install--upgrade pip-ро нависед. Ин PIP-ро ба версияи охирин навсозӣ мекунад.

Пас аз насб кардани Python ва PIP, шумо метавонед PIP-ро барои насб кардани бастаҳо аз PyPI истифода баред. Барои намуна:

pip install package-name

package-name номи бастаест, ки шумо насб кардан мехоҳед.

Фармонҳои PyPI

Фармон Вазифа 

pip install package-name

насб кардани баста

 

pip install package-name==version

насб кардани версияи бастаи мушаххас

 

pip install--upgrade package-name

навсозии баста

 

pip uninstall package-name

хориҷ кардани баста

 

pip search search-query

ҷустуҷӯи бастаҳо

 

pip list

намоиши рӯйхати бастаҳои насбшуда

 

pip list--outdated

ҷустуҷӯи бастаҳои кӯҳнашуда

 

pip show package-name

дидани маълумоти баста

 

pip freeze > requirements.txt
