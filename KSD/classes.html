Classes - Синфҳо

Объект - маҷмӯи маълумот (тағйирёбандаҳо) ва усулҳо (функсияҳо) мебошад. Синф - расми (нақшаи) объект аст.

Синфҳо

Барои сохтани синф дар Python, калимаи калидии class истифода бурда мешавад:


        class ClassName:
            # Муайянкунии синф

Дар ин ҷо мо синферо бо номи ClassName муайян кардем.

Мисол:


        class Bike:
            name = ""
            gear = 0

Дар ин ҷо:

  • Bike - номи синф;

  • name ва gear - тағирёбандаҳо дар дохили синф;

Тағирёбандаҳои дохили синф атрибутҳо номида мешаванд.

Объектҳо

Объект - нусхаи синф аст. Масалан, Bike синф аст ва мо метавонем аз ин синф объектҳо ба мисли bike1, bike2 созем.

Синтаксис барои сохтани объектҳои синф дар Python ин аст:


        objectName = ClassName()

Мисол:


        # Сохтани синф
        class Bike:
            name = ""
            gear = 0
        
        # Сохтани объекти синф
        bike1 = Bike()

Дар ин ҷо bike1 объекти синфи Bike мебошад. Мо метавонем ин объектро барои дастрасӣ ба атрибутҳои синфи Bike истифода барем.

Дастрасӣ ба атрибутҳои синф тавассути объектҳо

Оператори . барои дастрасӣ ба атрибутҳои синф истифода бурда мешавад. Масалан:


        # Тағир додани атрибути name-и объекти bike1
        bike1.name = "Mountain Bike"
        
        # Пайдо кардани дастрас ба атрибути gear-и объекти bike1 
        bike1.gear

Дар ин ҷо мо bike1.name ва bike1.gear-ро барои дастрасӣ ва тағир додани арзиши атрибутҳои name ва gear истифода бурдем.

Мисоли дигар:


        # Сохтани синф
        class Bike:
            name = ""
            gear = 0
        
        # Сохтани объекти синф
        bike1 = Bike()
        
        # Дастрасӣ ба атрибутҳои объект ва таъини арзишҳои нав
        bike1.gear = 11
        bike1.name = "Mountain Bike"
        
        # Намоиши маълумоти объект дар экран
        print(f"Name: {bike1.name}, Gears: {bike1.gear} ")

Натиҷа:


        Name: Mountain Bike, Gears: 11

Дар ин ҷо мо синфи Bike-ро бо ду атрибут муайян кардем: name ва gear. Мо объекти bike1-и синфи Bike сохтем, ба атрибутҳои объект дастрасӣ пайдо кардем ва сипас онҳоро бо истифода аз оператори . иваз кардем.

Сохтани якчанд объектҳои як синф

Мо метавонем объектҳои зиёди як синфро созем. Масалан:


        # Сохтани синф
        class Employee:
            # Муайян кардани атрибут 
            employee_id = 0
        
        # Сохтани ду объекти синфи Employee
        employee1 = Employee()
        employee2 = Employee()
        
        # Дастрасӣ ба атрибутҳои объекти employee1 ва таъини арзиши нав 
        employee1.employeeID = 1001
        print(f"Employee ID: {employee1.employeeID}")
        
        # Дастрасӣ ба атрибутҳои объекти employee2 ва таъини арзиши нав 
        employee2.employeeID = 1002
        print(f"Employee ID: {employee2.employeeID}")

Натиҷа:


        Employee ID: 1001
        Employee ID: 1002

Методҳо дар Python

Функсияи дар дохили синф муайяншуда метод номида мешавад.


        # Сохтани синф
        class Room:
            length = 0.0
            width = 0.0
            
            # Методи ҳисобкунии минтақа
            def calculate_area(self):
                print("Area of Room =", self.length * self.width)
        
        # Сохтани объекти синфи Room
        study_room = Room()
        
        # Таъин кардани арзишҳои нав ба ҳама атрибутҳои объект
        study_room.length = 42.5
        study_room.width = 30.8
        
        # Дастрасӣ ба метод дар дохили синф
        study_room.calculate_area()

Натиҷа:


        Area of Room = 1309.0

Дар ин ҷо мо синфи Room-ро бо атрибутҳои length ва width ва методи account_area() сохтем. Пас аз он мо як объекти study_room-и синфи Room сохтем ва ин объектро барои таъин кардани арзишҳо ба атрибутҳои length ва width истифода бурдем. Дар хотир доред, ки мо инчунин ин объектро барои даъват кардани метод дар дохили синф бо истифода аз оператори . истифода мебарем:


        study_room.calculate_area()

Конструкторҳо дар Python

Конструктор - методи махсусест, ки объекти синфро оғоз мекунад. Ин методест, ки ҳангоми сохтани як мисоли нави синф даъват мешавад.


        class Bike:
            # Функсияи конструктор    
            def __init__(self, name = ""):
                self.name = name
        
        bike1 = Bike()

Дар ин ҷо __init__() функсияи конструктор аст, ки ҳар вақте ки объекти нави ин синф сохта мешавад, даъват карда мешавад. Конструктор арзиши атрибути name-ро оғоз мекунад. Мо self.name-ро барои истинод ба атрибути name-и объекти bike1 истифода мебарем.

Агар мо конструкторро барои оғоз кардани арзишҳо дар дохили синф истифода барем, мо бояд ҳангоми сохтани объекти синф арзиши мувофиқро гузаронем:


        bike1 = Bike("Mountain Bike")

Дар ин ҷо, "Mountain Bike" ҳамчун аргументи name ба функсияи __init__() интиқол дода мешавад (ки конструктори синфи Bike аст) барои оғоз кардани атрибути name.
