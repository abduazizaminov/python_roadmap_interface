Generator Expressions - Ифодаҳои генераторӣ

Генератор дар Python - функсияест, ки итераторро бармегардонад, ки ҳангоми такрорӣ пайдарпайии арзишҳоро тавлид мекунад. Генераторҳо вақте муфиданд, ки ба мо лозим аст, ки пайдарпайии бузурги арзишҳоро ба даст орем, аммо мо намехоҳем, ки ҳамаи онҳоро дар як вақт дар хотира нигоҳ дорем.

Сохтани генератор дар Python

Мисли функсияҳои муқаррарӣ, функсияи генераторро дар Python бо истифода аз калимаи калидии def муайян кардан мумкин аст, аммо ба ҷои оператори return, оператори yield истифода мешавад.


        def generator_name(arg):
            # Корпуси генератор
            yield something

Дар ин ҷо калимаи калидии оператори yield барои баргардонидани арзиш аз генератор истифода мешавад. Вақте ки генератор даъват карда мешавад, корпуси он (код) фавран иҷро карда намешавад. Ба ҷои ин, объекти генератор баргардонида мешавад, ки онро барои тавлиди арзишҳо такрор кардан мумкин аст.

Мисоли функсияи генератор, ки пайдарпаии рақамҳоро тавлид мекунад:


        def my_generator(n):
        
            # Оғоз кардани ҳисобкунак
            value = 0
        
            # Давра то он даме, ки ҳисобкунак аз n камтар бошад, кор мекунад
            while value < n:
        
                # Баргардонидани арзиши ҳисобкунак
                yield value
        
                # Зиёд кардани ҳисобкунак
                value += 1
        
        # Иҷрои итератсияи генератор
        for value in my_generator(3):
        
            # Баровардани ҳар як арзиши аз генератор гирифташуда
            print(value)

Натиҷа:


        0
        1
        2

Дар ин ҷо функсияи генератори my_generator() адади бутуни n-ро ҳамчун аргумент мегирад ва пайдарпайии ададҳоро аз 0 то n-1 месозад. Калимаи калидии yield барои дарёфти арзиш аз генератор ва таваққуфи иҷрои вазифаи генератор то дархости арзиши навбатӣ истифода мешавад.

Сикли for тавассути объекти генератори аз ҷониби функсияи my_generator() сохташуда такрор мешавад ва оператори print ҳар як арзиши тавлидкардаи генераторро чоп мекунад. Мо инчунин метавонем объекти генераторро аз функсияи генератор бо даъвати он мисли ҳама гуна функсияи дигар созем:


        generator = my_range(3)
        print(next(generator))  # 0
        print(next(generator))  # 1
        print(next(generator))  # 2

Ифодаҳои генератор дар Python

Ифодаи генератор роҳи мухтасари сохтани объекти генератор мебошад. Он ба абстраксияи рӯйхат шабеҳ аст, аммо ба ҷои сохтани рӯйхат, он объекти генераторро эҷод мекунад, ки онро барои тавлиди арзишҳо такрор кардан мумкин аст.

Ифодаи генератор дар Python синтаксиси зерин дорад:


        (ифодаи for элемент in итератор)

Дар ин ҷо ифода арзишест, ки барои ҳар як унсури итератор баргардонида мешавад.

Ифодаи генератор як объекти генераторро эҷод мекунад, ки ҳангоми такрорӣ арзишҳои ифодаро барои ҳар як унсури итератор як-як ба вуҷуд меорад.

Биёед мисоли ифодаи генераторро дар Python дида бароем:


        # Сохтани объекти генератор
        squares_generator = (i * i for i in range(5))
        
        # Итератсия тавассути генератор ва баровардани арзишҳо
        for i in squares_generator:
            print(i)

Натиҷа:


        0
        1
        4
        9
        16

Дар ин ҷо мо объекти генераторро сохтем, ки ҳангоми такрорӣ ҳар як ададро дар пайдарпаӣ аз 0 то 4 мураббаъ мекунад. Сипас мо барои такрор кардани генератор ва чоп кардани арзишҳо сикли for-ро истифода мебарем.
