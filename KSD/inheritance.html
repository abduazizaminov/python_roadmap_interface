Inheritance - Мерос

Мерос ин равандест, ки дар он як синф хусусиятҳо ва усулҳои синфи дигарро мерос мегирад.

Зерсинф ва суперсинф

Синтаксиси мерос дар Python


        # Муайян кардани суперсинф
        class super_class:
            # Атрибутҳо ва методҳои суперсинф
        
        # Мерос
        class sub_class(super_class):
            # Атрибутҳо ва метод super_class
            # Атрибутҳо ва метод sub_class

Дар ин ҷо мо синфи sub_class аз синфи super_class мерос мегирем.

Мисол:


        class Animal:
            # Атрибут ва методи синфи асосӣ
            name = ""
            def eat(self):
                print("I can eat")
        
        # Мерос аз синфи Animal
        class Dog(Animal):
            # Методи нав дар синфи фаръӣ
            def display(self):
                # Дастрасӣ ба атрибути name синфи асосӣ бо истифода аз self
                print("My name is ", self.name)
        
        #  Сохтани объекти синфи фаръӣ
        labrador = Dog()
        
        # Дастрасӣ ба атрибут ва методи синфи асосӣ
        labrador.name = "Rohu"
        labrador.eat()
        
        # Даъват кардани методи синфи фаръӣ
        labrador.display()

Натиҷа:


        I can eat
        My name is Rohu

Дар ин ҷо мо аз синфи асосии Animal як синфи фаръи Dog-ро сохтем. Ба сатрҳои зерини код диққат диҳед:


        labrador.name = "Rohu"
        labrador.eat()

Дар ин ҷо мо labrador (объекти синфи Dog) барои дастрасӣ ба name ва eat() синфи Animal истифода мебарем. Ин имконпазир аст, зеро синфи фаръӣ тамоми атрибутҳо ва методҳои синфи асосиро мерос мегирад. Илова бар ин, мо ба атрибути name дар дохили методи синфи Dog бо истифода аз self дастрасӣ пайдо кардем.

Муносибатҳо дар мерос

Дар Python, мерос ин муносибат намуди is-a аст. Яъне, мо меросро танҳо дар он вақт истифода мебарем, ки дар байни ду синф муносибати "аст" вуҷуд дошта бошад. Барои намуна:

  • Мошин - воситаи нақлиёт аст.
  • Себ - мева аст.
  • Гурба - ҳайвон аст.

Мисол. Бисёркунҷа - фигураи пӯшидае мебошад, ки 3 ё зиёда тараф дорад. Фарз мекунем, ки мо синфи Polygon дорем, ки чунин муайян карда шудааст:


        class Polygon:
            def __init__(self, no_of_sides):
                self.n = no_of_sides
                self.sides = [0 for i in range(no_of_sides)]
        
            def inputSides(self):
                self.sides = [float(input("Enter side "+str(i+1)+" : ")) for i in range(self.n)]
        
            def dispSides(self):
                for i in range(self.n):
                    print("Side",i+1,"is",self.sides[i])

Ин синф дорои тағирёбандаҳо барои нигоҳ доштани адад (n) ва дарозӣ (дар шакли рӯйхате, ки sides номида мешавад) ҳар як паҳлӯи бисёркунҷаро дорад.

  • Методи inputSides() дарозии ҳар як тарафро мегирад.
  • Методи dispSides() дарозии тарафҳоро нишон медиҳад.

Секунҷа - бисёркунҷаест, ки се тараф дорад. Аз ин рӯ, мо метавонем синфи Triangle созем, ки аз синфи Polygon мерос мегирад. Ин ҳама атрибутҳои синфи Polygon-ро барои синфи Triangle дастрас мекунад. Ба мо лозим нест, ки боз атрибутҳо ва методҳоро муайян кунем. Масалан:


        class Triangle(Polygon):
            def __init__(self):
                Polygon.__init__(self,3)
        
            def findArea(self):
                a, b, c = self.sides
        
                # Ҳисоб кардани нимпериметр
                s = (a + b + c) / 2
                area = (s*(s-a)*(s-b)*(s-c)) ** 0.5
        
                # Намоиши минтақаи секунҷа
                print('The area of the triangle is %0.2f' %area)

Синфи Triangle дорои методи нави findArea() барои ҳисоб кардан ва намоиш додани майдони секунҷа мебошад.

Коди барнома:


        class Polygon:
            # Инитсиализатсияи шумораи тарафҳо
            def __init__(self, no_of_sides):
                self.n = no_of_sides
                self.sides = [0 for i in range(no_of_sides)]
        
            def inputSides(self):
                self.sides = [float(input("Enter side "+str(i+1)+" : ")) for i in range(self.n)]
        
            # Методи баровардани дарозии ҳар як тарафи бисёркунҷа
            def dispSides(self):
                for i in range(self.n):
                    print("Side",i+1,"is",self.sides[i])
        
        class Triangle(Polygon):
            # Инитсиализатсияи шумораи тарафҳо (3), бо даъвати методи __init__() синфи Polygon
            def __init__(self):
                Polygon.__init__(self,3)
        
            def findArea(self):
                a, b, c = self.sides
        
                # Ҳисоб кардани нимпериметр
                s = (a + b + c) / 2
        
                # Ҳисоб кардани минтақаи секунҷа бо формулаи Ҳерон
                area = (s*(s-a)*(s-b)*(s-c)) ** 0.5
                print('The area of the triangle is %0.2f' %area)
        
        # Сохтани объекти синфи Triangle
        t = Triangle()
        
        # Ворид кардани дарозии тарафҳои секунҷа
        t.inputSides()
        
        # Баровардани дарозии тарафҳои секунҷа
        t.dispSides()
        
        # Ҳисоб кардан ва баровардани минтақаи секунҷа
        t.findArea()

Натиҷа:


        Enter side 1 : 3
        Enter side 2 : 5
        Enter side 3 : 4
        Side 1 is 3.0
        Side 2 is 5.0
        Side 3 is 4.0
        The area of the triangle is 6.00

Методи super() дар мерос

Методи super() дар Python объекти проксиро (объекти муваққатии суперкласс) бармегардонад, ки ба шумо имкон медиҳад ба методҳои синфи асосӣ дастрасӣ пайдо кунед.


        class Animal:
            # Атрибутҳо ва методи синфи асосӣ
            name = ""
            def eat(self):
                print("I can eat")
        
        # Мерос аз синфи Animal
        class Dog(Animal):
        
            # Аз нав таъин кардани методи eat()
            def eat(self):
                
                # Фарёд кардани методи eat()-и синфи асосӣ бо методи иловагии super()
                super().eat()
                
                print("I like to eat bones")
        
        # Сохтани объекти синфи фаръӣ
        labrador = Dog()
        
        labrador.eat()

Натиҷа:


        I can eat
        I like to eat bones

Дар ин ҷо методи eat()-и синфи фаръӣ Dog ҳамон методи синфи асосии Animal-ро аз нав таъин мекунад.

Дар дохили синфи Dog, мо сатри зерини кодро барои даъват кардани методи eat()-и синфи асосии Animal аз синфи фаръӣ Dog истифода мебарем:


        # Даъват кардани методи eat()-и синфи асосӣ бо истифода аз методи иловагии super() 
        super().eat()

Ҳамин тавр, вақте ки мо методи eat()-ро бо истифода аз объекти labrador даъват мекунем:


        labrador.eat()

Ҳам версияи аз нав таъин шудаи методии eat()-и синфи фаръӣ ва ҳам версияи синфи асосии методи eat() иҷро мешаванд.
