Conda

Conda - платформа барои идоракунии бастаҳо, муҳитҳо ва вобастагӣ дар Python ва дигар забонҳои барномасозӣ мебошад. Conda Python роҳи осони идоракунии насбҳо ва навсозиҳои бастаҳо ва эҷод ва идора кардани муҳити виртуалии Python барои ҷудо кардани лоиҳаҳо ва вобастагии онҳоро таъмин мекунад.

Насб кардани Conda Python

Барои насб кардани Conda Python, ин қадамҳоро иҷро кунед:

  1. Насбкунандаи Miniconda ё Anaconda-ро зеркашӣ кунед:

    • Miniconda маҷмӯи ҳадди ақали абзорҳои Conda ва Python-ро таъмин мекунад, дар ҳоле ки Anaconda бастаҳои зиёди қаблан насбшударо барои ҳисоббарории илмӣ ва таҳлили додаҳо дар бар мегирад.

    • Шумо метавонед насбкунандаи Miniconda ё Anaconda-ро аз вебсайти расмии Continuum.io зеркашӣ кунед.

  2. Насбкунандаро иҷро кунед:

    • Насбкунандаеро, ки шумо зеркашӣ кардаед, иҷро кунед.

    • Дастурҳои насбкуниро риоя кунед. Роҳи насб ва имконоти дигарро мувофиқи хоҳиши худ интихоб кунед.

  3. Насбро тафтиш кунед:

    • Пас аз ба итмом расидани насб, терминал ё сатри фармонро кушоед.

    • conda--version-ро ворид кунед, то боварӣ ҳосил кунед, ки Conda дуруст насб шудааст.

Ҳоло шумо бояд Conda Python-ро дар компютери худ насб кунед ва шумо метавонед онро барои идоракунии бастаҳо ва муҳити Python истифода баред.

Фармон Вазифа

conda install

насб кардани баста

conda remove

хориҷ кардани баста

conda search

ҷустуҷӯи баста

conda list

рӯйхати бастаҳои насбшуда

conda update--all

навсозии Conda ва ҳама бастаҳо

conda create

сохтани муҳити нав

conda activate

фаъол кардани муҳити зист

conda deactivate

хомӯш кардани муҳити ҷорӣ

conda env list

рӯйхати ҳама муҳитҳо

conda env remove

хориҷ кардани муҳити зист

conda env export

содироти муҳити атроф ба файл

conda env create-f

сохтани муҳити атроф аз файли муҳити зист
