PIP

PIP - менеҷери бастаҳо мебошад. Он ба шумо имкон медиҳад, ки бастаҳоро дар Python насб ва идора кунед.

Насб кардани PIP барои Python

Агар PIP насб карда нашуда бошад, шумо бояд скриптро зеркашӣ кунед. Папкаро бо рақами версияи Python-и худ интихоб кунед. Пас аз он, дар сатри фармон ба ҷузвдон бо скрипт равед ва фармонро иҷро кунед:


        python get-pip.py

Агар PIP дар папкаи ProgramFiles насб карда шуда бошад, шумо бояд сатри фармонро ҳамчун администратор иҷро кунед.

Агар шумо PIP-ро насб карда бошед, вале дар консол хатогӣ пайдо кунед, шумо бояд роҳро ба файли pip.exe дар тағирёбандаи Path муқаррар кунед. Барои ин, ба "Свойства компьютера">"Дополнительные параметры системы">"Переменные среды" гузаред. Дар ин ҷо (вобаста ба версияи Windows) ё роҳро ба роҳи мавҷуда, ки бо нуқта-вергул ҷудо карда шудааст, илова кунед ё "Создать"-ро пахш кунед ва роҳро ба соҳаи нав нусхабардорӣ кунед.

Фармонҳои PIP

Фармон Вазифа

pip install package-name

версияи охирини бастаро насб мекунад;

pip install package-name==4.8.2

версияи бастаи 4.8.2-ро насб мекунад;

pip install package-name--upgrade

версияи бастаро нав мекунад;

pip download

бастаҳоро зеркашӣ мекунад;

pip uninstall

бастаҳоро хориҷ мекунад;

pip freeze

рӯйхати бастаҳои насбшударо дар формати зарурӣ намоиш медиҳад (одатан барои навиштан дар requirements.txt истифода мешавад);

pip list

рӯйхати бастаҳои насбшударо намоиш медиҳад;

pip list–outdated

рӯйхати бастаҳои кӯҳнашударо намоиш медиҳад;

pip show

маълумотро дар бораи бастаи насбшуда нишон медиҳад;

pip check

бастаҳои насбшударо барои мутобиқати вобастагӣ тафтиш мекунад;

pip search

бо номи воридшуда бастаҳои дар PyPI нашршударо ҷустуҷӯ мекунад;

pip wheel

мувофиқи талабот ва вобастагии шумо архиви wheel-ро ҷамъ мекунад;

pip hash

хэшҳои архиви бастаҳоро ҳисоб мекунад;

pip completion

фармони ёрирасон барои анҷом додани фармони асосӣ истифода мешавад;

pip help

оид ба фармонхо ёрӣ мерасонад.
