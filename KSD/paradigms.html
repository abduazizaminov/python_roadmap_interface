Paradigms - Парадигмаҳо

Парадигмаи барномасозӣ - услуб ё равиши бунёдии навиштани рамзи нармафзор аст, ки муайян мекунад, ки чӣ гуна таҳиягарон метавонанд мушкилотро ҳал кунанд ва коди худро ташкил кунанд. Ҳар як парадигма принсипҳо, усулҳо ва воситаҳои худро барои таҳияи нармафзор пешниҳод мекунад. Парадигмаҳои асосии барномасозӣ иборатанд аз:

Тавсиф

Ташкили барнома дар атрофи объектҳое, ки бо ҳамдигар амал мекунанд

Синфҳо, объектҳо, мерос

Барномасозии функсионалӣ

Ташкили барнома дар атрофи функсияҳо ва таркибҳои онҳо

Функсияҳои тартиботи олӣ, бастаҳо

Муайян кардани пайдарпаии дастурҳо барои иҷрои барнома

Дастурамалҳо, сиклҳо, шартҳо

Барномасозии проседурӣ

Барномасозии проседурӣ. Дар ин парадигма, барнома ба маҷмӯи проседураҳо ё функсияҳое тақсим мешавад, ки вазифаҳои мушаххасро иҷро мекунанд. Проседураҳо барои иҷрои барнома пайдарпай даъват карда мешаванд.

Мисол:


        def calculate_area(length, width):
            area = length * width
            return area

        result = calculate_area(5, 4)
        print("Масоҳати росткунҷа:", result)

Натиҷа:


            Масоҳати росткунҷа: 20

Барномасозии ба объект нигаронидашуда (БОН)

Барномасозии ба объект нигаронидашуда (БОН). Ин парадигма ба мафҳуми объектҳо, ки намунаҳои синфҳо мебошанд, асос ёфтааст. Ҳар як объект дорои хосиятҳо (майдонҳо) ва усулҳо (функсияҳо) мебошад, ки метавонанд ҳолати худро тағир диҳанд ё бо дигар объектҳо ҳамкорӣ кунанд.

Мисол:


        class Rectangle:
            def __init__(self, length, width):
                self.length = length
                self.width = width
            
            def calculate_area(self):
                return self.length * self.width

        rectangle = Rectangle(5, 4)
        result = rectangle.calculate_area()
        print("Масоҳати росткунҷа:", result)

Натиҷа:


            Масоҳати росткунҷа: 20

Барномасозии функсионалӣ

Барномасозии функсионалӣ. Дар ин ҷо барномаҳо аз функсияҳое сохта мешаванд, ки моликияти мустақил будан аз маълумоти беруна ва таъсири тарафро доранд. Функсияҳо ҳамчун "объектҳои дараҷаи аввал" баррасӣ карда мешаванд, ки маънои онро дорад, ки онҳо метавонанд ҳамчун аргумент интиқол дода шаванд, аз дигар функсияҳо баргардонида шаванд ва дар тағирёбандаҳо нигоҳ дошта шаванд.

Мисол:


        def calculate_area(length, width):
            return length * width

        result = calculate_area(5, 4)
        print("Масоҳати росткунҷа:", result)

Натиҷа:


            Масоҳати росткунҷа: 20

Барномасозии императивӣ

Барномасозии императивӣ. Ин равишест, ки дар он барнома аз дастурҳое иборат аст, ки ба компютер чӣ гуна иҷрои вазифаро нишон медиҳанд. Дар ин ҷо барномасоз қадамҳоеро тавсиф мекунад, ки барои ба даст овардани натиҷаи дилхоҳ бояд риоя шаванд.

Мисол:


        numbers = [1, 2, 3, 4, 5]
        total = 0

        for num in numbers:
            total += num

        print("Ҷамъи рақамҳо:", total)

Натиҷа:


            Ҷамъи рақамҳо: 15

Барномасозии декларативӣ

Барномасозии декларативӣ. Барнома тасвир мекунад, ки чӣ кор кардан лозим аст, на ин ки чӣ тавр бояд кард. Дар ин парадигма, барномасоз натиҷаи дилхоҳро муайян мекунад, на пайдарпаии қадамҳо барои ноил шудан ба он.

Мисол:


        numbers = [1, 2, 3, 4, 5]
        squares = [num ** 2 for num in numbers]
        print("Мураббаъҳои рақамҳо:", squares)

Натиҷа:


            Мураббаъҳои рақамҳо: [1, 4, 9, 16, 25]
